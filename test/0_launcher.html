🛡️ 웹보안관 시연 런처

아래 링크에 마우스를 0.5초 올려두면 익스텐션이 분석을 시작합니다

사용 방법
1. python -m http.server 3000 으로 로컬 서버 실행
2. http://localhost:3000/0_launcher.html 접속
3. 각 링크에 마우스를 올려서 툴팁 확인
