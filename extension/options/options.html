API 서버 모드 설정

로컬 / 원격 서버 전환 로컬 개발용 API 서버(localhost)를 사용합니다.
현재 API 주소 http://localhost:8000
원격 서버 IP 입력

로컬 서버 설치 및 구동 방법

로컬 컴퓨터에서 보안 판정 및 AI 분석 서버를 직접 실행할 수 있습니다.

1
로컬 서버 프로그램 다운로드

아래 버튼을 눌러 로컬 서버 구동용 소스코드를 다운로드하고 압축을 풉니다.

📥 로컬 서버 (ZIP) 다운로드
2
서버 실행 프로그램 실행

압축을 푼 폴더의 backend/start_server.bat 파일을 더블 클릭하여 실행합니다.

💡 Python 3.9 이상 환경이 필요하며, 최초 실행 시 필요한 패키지가 자동으로 설치됩니다.
3
로컬 모드로 전환

위의 설정에서 로컬 / 원격 서버 전환 토글을 해제(비활성화)하면 localhost:8000으로 자동 연결됩니다.
